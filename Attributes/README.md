## Objective

Build an application that discovers and executes methods based on custom attributes.

## 📋 Requirements

- [x]  Define a custom attribute (e.g., `[Runnable]`)
- [x]  Create several classes with methods decorated with the `[Runnable]` attribute
- [x]  Use reflection to scan the current assembly for methods marked with `[Runnable]`
- [x]  Invoke the discovered methods dynamically and display their outputs
