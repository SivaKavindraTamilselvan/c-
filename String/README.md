## Objective

Write a program to manage a list of strings (e.g., names or tasks).

## 📋 Requirements

- [x]  Use a `List<string>` to store items
- [x]  Allow the user to add, remove, and display items
- [x]  Utilize loops and basic string methods (like `Trim()`, `ToUpper()`) to process user input
